# Muistilappu verkkosivu

## Mitä tällä voi tehdä?
 1. Tehdä muistilappuja
 2. Liikuttaa muistilappuja
 3. Tallentaa muistilappuja tietokantaan
 4. Kirjaudu sisään/Rekisteröidy
 5. Muistilaput tallentuu per käyttäjä
